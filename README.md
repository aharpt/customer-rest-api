# Customer REST API

## Overview
This is a REST API project that uses Spring Boot to allow for basic CRUD operations for a Customer/Product/Orders system. These APIS are secured via Spring Security and JWT. The system data is stored in a PostgresSQL database that runs within Docker.

## Important Links:
- API Base URL: https://customer-rest-api-production.up.railway.app/api/v1/
- Swagger Docs UI: https://customer-rest-api-production.up.railway.app/swagger-ui/index.html

## Technologies Used:
- Spring Boot
- Spring Security
- Maven
- PostgresSQL
- Docker
